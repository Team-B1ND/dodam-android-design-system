## DUI는?
DUI는 Dodam UI의 합성어로 본 Dodam Android Design System를 개발합니다.</br>
본 프로젝트는 도담도담 안드로이드 디자인 시스템입니다. 도담도담 티쳐와 DAuth의 일관된 UI 적용을 위하여 개발되고 있습니다.</br>
